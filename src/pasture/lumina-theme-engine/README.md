## Lumina Theme Engine
                                                                                                                                                                                                                                                                               
This is a theme engine for the Lumina Desktop
                                                                                                                                                                                                                                                                               
### Dependencies                                                                                                                                                                                                                                                       
                                                                                                                                                                                                                                                                               
1. Qt 5.4+
  - qtbase >= 5.4.0
  - qtsvg >= 5.4.0
  - qttools >= 5.4.0

2. Lumina Desktop

Environment Variable QT_QPA_PLATFORMTHEME=lthemeengine needs to be set
