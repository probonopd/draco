# Draco Desktop Environment

![screenshot](screenshot-01.jpg)


A simple desktop environment based on Lumina. **This is work-in-progress, not for end-users (yet).**

## Requirements

  * Qt 5.2+
    * QtWidgets
    * QtNetwork
    * QtConcurrent
    * QtX11Extras
    * QtDBus
  * xcb
    * xcb-xinput
    * xcb-randr
    * xcb-ewmh
    * xcb-icccm
    * xcb-image
    * xcb-composite
    * xcb-damage
    * xcb-util
    * xcb-atom
  * xdamage
  * xscreensaver
  * powerkit 1.x
  * qtfm 6.2.x
  * hicolor-icon-theme
  * adwaita-icon-theme (or compatible)
  * xrandr
